Introduction

The task was to synchronously move motors on two micro zed boards. When motor on first desk is moved, then through packet new position is sent to second board and it moves motor to same position as first board. Aplication is real time, that means motion delay from moving first motor to move of second motors should be bounded.

Running and Compilation

The package is composed of 2 independent programs: Tire for board that is being moved and Wheel when moved sends information about new position. Starting point for both programs is main function in main.c files in directories with other * source and header files. To compile programs library xlnx_zynq7k.h must be linked and then both programs are compiled independently simply by linking all .h files in tire or wheel directory.
